# Social-Media-Data-Analysis-using-MySQL
A MySQL-based social media analytics project featuring users, posts, likes, and comments tables. Demonstrates SQL joins, relationships, filtering, aggregation, and engagement analysis using real-world database concepts.

Social Media Data Analysis using MySQL
This project is a relational database management system built using MySQL to analyze social media platform activities. The database contains four major tables:

Users – Stores user details such as username, email, joining date, and country.
Posts – Contains post information including content type and posting date.
Likes – Tracks user likes on different posts.
Comments – Stores comments made by users on posts.

The project demonstrates practical implementation of:

Database normalization
Table relationships using foreign keys
SQL joins
Aggregate functions
Data filtering and sorting
User engagement analysis
Content performance tracking
